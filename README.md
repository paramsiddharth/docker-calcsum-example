# Sum Calculator
A simple programme that calculates the sum of two numbers.

## Compilation
``` bash
gcc sumcalc.c -o sumcalc
```

## Executiom
``` bash
./sumcalc
```

## Building Docker image
``` bash
docker build -t sumcalc .
```

## Executing the image in a container
``` bash
docker run --rm -ti sumcalc
```
